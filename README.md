# bakery-items project

This is my semester project for bakery management. It has an admin dashboard where we can manage all bakery items.

## Technologies Used
* HTML
* CSS
* JavaScript
* JSON Server (for database)

## Main Features
* Can add new bakery items with auto increment id
* Can edit and update product details like name and price
* Can delete items from the table
* Dashboard shows total counts and categories like cakes and breads
